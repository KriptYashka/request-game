PixelBattleQuestion.

Ищи вопросы, находи ответы и захватывай территорию. Для этого потребуются следующие команды:
/ — главная страница
/question/id_число — вопрос (GET-метод) и ответ (POST).
В POST передавайте ответ через параметр 'answer'. Тип не важен, т.к. будет проверятся, как строка.

{
"answer": "SHP_THE_BEST!!1!"
}
